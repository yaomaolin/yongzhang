业务流程闭环

培训计划生命周期

新建计划 → 选择培训对象 → 发布/存草稿 → 报名中 → 进行中 → 签到签退 → 赋分 → 已结束

参训人员流程

浏览培训 → 报名 → 签到 → 参加培训 → 签退 → 上传材料 → 申报学分 → 等待审核 → 学分入账

学分审核流程

学员申报 → 管理员审核（弹窗赋分 / 驳回填理由）→ 通过：学分入账 / 驳回：学员重新申报

外部系统关联

签到签退 → 跳转签到系统（超链接示意）
选人组件 → 组织架构树选人弹窗（搜索 + 学院/角色筛选）
